# Evomart
Evomart is a platform designed for buying and selling farm lands and properties, facilitating seamless communication between buyers and sellers.

## Usage
Evomart allows users to:
- Browse available farm lands and properties for sale.
- List their own properties for sale.
- Contact sellers to inquire about properties.

## Prerequisites
Before running the project, ensure you have the following software installed:
- [Node.js](https://nodejs.org/): Download and install Node.js, which includes npm.

## Installation
1. Clone the repository: $ git clone https://github.com/YOUR_USERNAME/PROJECT_TITLE
2. Install dependencies: npm install 
3. Start the Development server : npm run dev (http://localhost:5173/)
4. Make build : npm run build

## Dependencies
- Material UI : For Components
- Formik : For form handling
- Yup : For client side validation for form
- Axios : For fetching Apis
